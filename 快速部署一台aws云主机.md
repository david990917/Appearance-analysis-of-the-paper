# 快速部署一台aws云主机

1. aws 官网启动机器

2. [Linux上面安装 Miniconda](https://docs.conda.io/projects/conda/en/latest/user-guide/install/linux.html)

   ```bash
   bash Miniconda3-latest-Linux-x86_64.sh
   ```

   添加路径

   ```
   export PATH="/home/ubuntu/miniconda3/bin/conda:$PATH"
   
   source ~/.bashrc
   ```
